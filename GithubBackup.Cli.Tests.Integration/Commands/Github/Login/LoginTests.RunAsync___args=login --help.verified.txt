Description:
  Authenticate with GitHub. Tokens are resolved from --token, GITHUB_BACKUP_TOKEN, or GitHub device flow.
  Device flow tokens are cached temporarily while valid. 

Usage:
  ghb login [options]

Options:
  -t, --token <token>                                                    GitHub token to use for this command.
                                                                         Takes precedence over GITHUB_BACKUP_TOKEN, temporary cache, and device flow.
  -d, --device-flow-auth                                                 Start interactive GitHub device flow authentication.
                                                                         Bypasses the temporary token cache. Requires a browser.
  -?, -h, --help                                                         Show help and usage information
  -q, --quiet                                                            Suppress non-essential status output.
  -l, --log-file <log-file>                                              Path to the log file.
  -v, --verbosity <Critical|Debug|Error|Information|None|Trace|Warning>  Set the verbosity level of the log file. [default: Information]
